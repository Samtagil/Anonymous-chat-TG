# Скриншоты для README

Положите в эту папку три PNG (рекомендуемая ширина 800–1200 px):

| Файл | Что снять |
|------|-----------|
| `01-chat-code.png` | Сообщение в общей ленте с видимым **CODE** и ником |
| `02-notifications.png` | Уведомление от бота (ответ, упоминание, активность) |
| `03-ai-helper.png` | Ответ или подсказка **AI-помощника** в диалоге |

После добавления файлов картинки появятся в основном [README.md](../README.md).
